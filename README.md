# DouglasTech

DouglasTech é uma empresa fictícia de tecnologia criada para demonstrar um site/portfólio. Este repositório contém o código do site institucional da DouglasTech e serve como exemplo de projeto front-end simples.

## Sobre

DouglasTech oferece soluções imaginárias em desenvolvimento web, design de interfaces e consultoria tecnológica. Todos os serviços, clientes e projetos mencionados neste repositório são fictícios e usados apenas para fins de demonstração.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
